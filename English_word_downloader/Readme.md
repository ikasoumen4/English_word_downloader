辞書によってダウンロードを優先するサイトを指定することができます。

url.txtに登録されている優先順にダウンロードが試行され、見つからなかった場合error.logにダウンロードできなかった言葉が出力されます。
また、ダウンロードエラーの前に例外単語処理を行いその単語独自の例外処理を定義することができます。
